Ejercicio #1

Fix it

Este ejercicio es sobre tu manejo de las herramientas de desarrollo de un browser. Recomendamos que instales Chrome y que uses Chrome Devtools. Si no sabés lo que es, podés familiarizarte con la herramienta en este link.

El ejercicio se trata sobre un autocomplete que está roto. Tu misión es encontrar los errores, arreglarlos y contarnos cuáles fueron tus pasos. ¿Qué cosas (fuera de cuestiones de estilo) harías para mejorarlo?
